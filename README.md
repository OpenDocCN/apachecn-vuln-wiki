## 下载

### Docker

```
docker pull apachecn0/flygon-vuln-wiki
docker run -tid -p <port>:80 apachecn0/flygon-vuln-wiki
# 访问 http://localhost:{port} 查看文档
```

### NPM

```
npm install -g flygon-vuln-wiki
flygon-vuln-wiki <port>
# 访问 http://localhost:{port} 查看文档
```
